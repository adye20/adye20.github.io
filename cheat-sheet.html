Ruby Enumerables

The enumberable module gives objects of collection classes additional collection-specific behaviors. Enumerables are a group of methods that traverse and search collection objects. Enumberables can sort, search, evaluate, or modify each member according to block of code.

Method Description Example
Each Calls block of code once for each element and passes that element as parameter array.each {|break| puts break}
Map Applies a given block for each item in self. Creates a new array with values returned by the code block (use map! if you want to replace your existing array). array.map {|square| square * square}
Find All Returns an array with all elements which the code block returns a true value array.find_all {|num| num % 2 == 0}
Sort Returns a new array with the elements sorted. array.sort
Shuffle Returns a new array with the elements of self randomized. array.shuffle
